# iPadOS Design Source

Source copy of the `ipados-design` skill.

Use this folder when you want the raw skill content independent of Codex or Claude packaging. The packaged copies live at:

- `codex/plugins/ipados-design/skills/ipados-design/`
- `claude/.claude/skills/ipados-design/`

The skill focuses on adapting iPhone-quality app design to iPadOS without stretching iPhone screens. It covers adaptive layout, NavigationSplitView, sidebars, inspectors, resizable windows, Stage Manager, pointer/trackpad, hardware keyboard, Apple Pencil, drag and drop, menu bars, accessibility, and iPad screenshot verification.
