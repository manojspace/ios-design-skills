# Codex iOS Design Plugin

Git-backed Codex marketplace for the `ios-design` plugin.

## Install

```bash
codex plugin marketplace add manojspace/codex-ios-design-plugin --ref main
codex plugin add ios-design@manojspace
```

## Update

```bash
codex plugin marketplace upgrade
codex plugin add ios-design@manojspace
```

Start a new Codex thread after installing or updating so Codex picks up the skill.
