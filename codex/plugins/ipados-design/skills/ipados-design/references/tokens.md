# Tokens

iPadOS should share the iPhone app's identity tokens, then adapt density and layout tokens for a larger canvas.

## Shared With iPhone

- Brand and accent colors.
- Text semantic colors.
- Typography roles.
- Semantic colors.
- State treatments.
- Motion language.

## iPad Additions

Add tokens for:

- Sidebar width.
- Content max width.
- Inspector width.
- Grid column minimum width.
- Panel spacing.
- Toolbar height and item spacing.
- Popover and sheet widths.

## Rules

- Do not increase every font size just because the screen is larger.
- Use density and hierarchy, not oversized typography, to fill space.
- Keep readable line length.
- Use monospaced digits for counts, times, prices, durations, and stats.
- Ensure dark mode and increased contrast work in multi-panel layouts.
