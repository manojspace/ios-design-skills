# Claude Project Instructions

For iOS UI work, use the project skill at `.claude/skills/ios-design/SKILL.md`.

For iPadOS UI work, use the project skill at `.claude/skills/ipados-design/SKILL.md`.

Apply it when working on user-visible SwiftUI/UIKit screens, components, layout, visual polish, accessibility, Dynamic Type, motion, haptics, simulator screenshots, or UI critique.

Do not apply it for pure networking, persistence, build config, CI, App Intents, or business logic unless the task changes user-visible UI behavior.
